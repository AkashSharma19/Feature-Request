# Coach — Feature Request Management Platform

Coach is a premium, enterprise-grade feature request and roadmap management system designed for high-performing product teams. It bridges the gap between user feedback and engineering execution with a sleek, glassmorphic interface inspired by industry leaders like Jira and Productboard.

## 🚀 Key Features

### 📊 Premium Dashboard
- **Glassmorphic UI**: A stunning, modern interface with smooth gradients and interactive elements.
- **Smart Filtering**: Filter requests by status, category, organization, or search query.
- **Summary Analytics**: Real-time stats on total, open, in-progress, and completed features.

### 🗺️ Interactive Roadmap
- **Drag-and-Drop Workflow**: Seamlessly move features across the roadmap to update their status (Admin only).
- **Quarterly Planning**: Organize your vision into clear, actionable quarterly blocks (Q1–Q4).
- **Progress Tracking**: Visual indicators of development progress for every feature.

### ✍️ Advanced Request Creation
- **Smart Duplicate Detection**: Real-time detection of similar requests as you type to prevent fragmentation.
- **TipTap Rich Text Editor**: A professional editor for feature descriptions with support for Bold, Italic, Lists, and Quotes.
- **Image Attachments**: Upload local images directly into descriptions via Base64 encoding.

### ✉️ Email Management System
- **Automated Workflows**: Pre-defined triggers for status updates, mentions, and action requirements.
- **Template Gallery**: A library of beautiful, high-end email templates with a live browser-preview canvas.
- **Admin Control**: Centralized management of user communications and branding.

### 🤖 Coach Support Bot
- **Interactive Assistance**: A floating AI-powered bot to help users navigate the platform and find information.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Rich Text**: [TipTap](https://tiptap.dev/)
- **Drag & Drop**: [@dnd-kit](https://dnd-kit.com/)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
